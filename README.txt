# ✊ Rock Paper Scissors

A simple browser-based Rock Paper Scissors game built with JavaScript, HTML, and CSS.

---

### 🎮 What It Is

- A fun, interactive game where you play Rock, Paper, or Scissors against the computer  
- Tracks scores in real time  
- Built to practice JavaScript fundamentals like event handling and DOM manipulation  
- Clean, responsive design that works on desktop and mobile

### 🚀 How to Use

1. Open `index.html` in your browser  
2. Click Rock, Paper, or Scissors to make your move  
3. See the computer’s choice and the round winner  
4. Keep track of scores and have fun!

---

### 📌 Why I Built It

- To strengthen my JavaScript skills with a real project  
- To practice handling user input and updating the UI dynamically  
- To build a simple but complete interactive game from scratch
